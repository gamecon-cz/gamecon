Program Iniciativa


 

Tento program slouží pro počítání iniciativy a akcí v rozšířeném soubojovém systému. Krátký popis funkcí programu je uveden spolu se spustitelným souborem v archívu. Autorem programu Michal Holub, jemuž také směřujte všechny komentáře a připomínky.

Program běží pod Windows 9x či Windows NT 3.51 nebo vyšší. K ovládání programu je nutná klávesnice a myš.

Program je freeware a můžete si jej stáhnout zde (velikost je 201k).
