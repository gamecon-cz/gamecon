About the server www.altar.cz

On this page you can find some basic information about the pages stored here, their layout, and the way they were created.

First, you can find here the list of browsers these pages are optimized for (in the order given here):

  1. Microsoft Internet Explorer 4
  2. Netscape Communicator 4

If you do not have one of the browsers in the list you can still view these pages, but you will have to take into account that they will not look exactly the the way we want to. In that case you can download one of the browsers; you can click the link in the browser list.

If you are not sure which browser and/or version you are using, or you are using a browser not listed and want to make sure these pages are displayed correctly, you can use the following list. It contains several hints about the correct look of these pages.

  • The background is dark blue, the rectangle containing this text being an exception; the text is dark blue on a nearly white texture.
  • The left column contains list of links, all of them rendered using inline images consisting of large yellowish capital letters.
  • The title of this page (About the server ...) is rendered in large yellow font.
  • The browser list is numbered, the font color being black. The numbers are indented by about five millimeters from the left border. There is no vertical space above the browser list (between the list and the paragraph above).
  • This list is bulleted, the bullets should be about ten millimeters from the left border.
  • The links in the list two paragraphs below are rendered using the correct color and font style. The correct properties for each link are given in the list.

Second, there is a list of all the hypertext link types you can run across when browsing through our server. The links are rendered using different colors and font styles (bold, italics) according to the type of their target document. The list follows. All the links in it lead to this page for they are here only for demonstration purposes.

  • Most of the links use light blue color. These links lead to a page on our server, different from the source one.
  • Violet links lead to another part of the source page.
  • Green links allow you to send e-mails. By clicking them, a window allowing you to compose a message is usually activated. However these links are handled in a different way in different browsers.
  • Links rendered in italics lead to a download of a non-text document, usually a .ZIP archive.
  • Black links lead to a different server than www.altar.cz.

The colors and font styles will not usually be displayed correctly when using a browser not given in the list on the top of this page. The color may also vary when you have already visited the target page.

If you have any problem, question, or proposal, you can send it to us using this fill-in form. If there is a problem making the form unusable, you can send an e-mail to the address Michal@altar.cz. If the form is in order, we prefer using it instead of the mail.
