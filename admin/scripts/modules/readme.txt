Soubory .php z tohoto bloku se nativně načítají do administrace jako moduly (kartičky nahoře). Uvnitř každého modulu musí být v úvodním komentáři určeno právo limitující přístup a název modulu, který se strojově načítá.

Pokud má modul víc submodulů, vytvoří se složka a do ní se dá hlavní modul stejného jména (jako ./aktivity/aktivity.php). Submoduly se načítají jako .php soubory od tamtéž.
